# TrustFlow

TrustFlow is a protocol that transforms social trust into a programmable funding primitive. Inspired by the Federated Byzantine Agreement model in Stellar, TrustFlow maps relationships between users as a trust graph, where each node defines its own trusted peers. Funding is then distributed not just directly, but across trust proximity—enabling mechanisms like “fund everyone within two degrees of trust” or “prioritize contributors trusted by my network.”

TrustFlow allows continuous, real-time allocation of funds based on evolving trust relationships. This creates a dynamic funding ecosystem where reputation and trust directly influence capital flow, making it ideal for open-source funding, DAOs, and public goods ecosystems.
